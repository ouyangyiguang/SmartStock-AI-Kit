# SmartStock-AI-Kit
**AI-Powered Stock Monitoring All-in-One Kit**
一款类“小智”风格的 AI 智能盯盘软硬全套件

---

## 📖 项目简介
本仓库开源了一套完整的 **AI 股票盯盘解决方案**。它不仅仅是一个软件，更是一个能够摆在你桌面上的“智能金融终端”。

通过自研的硬件外壳设计 + 嵌入式系统 + **LLM（大语言模型）**分析，它能像“小智”一样实时捕捉市场动态，不再让你错过每一个关键的买卖点。

## ✨ 核心特性
* **💡 智能唤醒与语音交互**：支持关键词唤醒，你可以直接问它：“今天茅台走势如何？”
* **📊 实时数据可视化**：通过高刷墨水屏/TFT屏，实时展示分时图、K线及资金流向。
* **🤖 AI 决策辅助**：集成 DeepSeek/GPT-4o 接口，自动总结盘中异动，分析龙虎榜数据。
* **⚡ 毫秒级预警**：支持自定义股价/成交量预警，通过硬件蜂鸣器或手机推送到位。
* **🛠️ 全栈开源**：
    * **硬件**：提供 PCB 原理图、3D 打印外壳模型。
    * **固件**：基于 ESP32 / 树莓派的 C++/Python 源码。
    * **后端**：基于 Node.js/Python 的数据爬取与 AI 处理网关。

## 🛠️ 硬件清单
* **主控**：ESP32-S3 / Raspberry Pi Zero 2W
* **显示**：2.13寸墨水屏 / 1.9寸 LCD 屏幕
* **外壳**：精美 3D 打印分体式设计

## 🚀 快速开始
1. **获取源码**：`git clone https://github.com/[your-username]/SmartStock-AI-Kit.git`
2. **硬件组装**：参考 `/hardware` 目录下的电路图完成 PCB 焊接与组装。
3. **烧录固件**：将 `/firmware` 代码烧录至主控板。
4. **服务配置**：配置 `config.json` 中的 API Key，启动你的私人 AI 交易员。
